האפליקציה מתחברת ישירות ל-API של צבע אדום. Poll כל 3 שניות, תגובה מיידית, 60+ ערים עם קואורדינטות.

how it works

איך זה עובד

01
API Polling
שירות הרקע מבצע בקשה ל-tzevaadom API כל 3 שניות
02
Parse + Map
מנתח ערים, ממפה לקואורדינטות, מסנן לפי הגדרות המשתמש
03
Notify
מפעיל נוטיפיקציה אדומה + צליל + רטט + מסך נעילה
04
Animate Map
מציג אנימציית טיל חיה על המפה עם מסלול ויירוט
05
Update All
מעדכן רשימה, ניתוח נתונים, בוט AI וצ'אט קהילתי
